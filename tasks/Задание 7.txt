Задание 7.

Срок исполнения: 19.11 - 26.11

Оформление исхлодного текста программы для обработки зрительных данных в виде
набора классов, осуществляющих отдельные операции обработки.
Потребуется использовать задания 5 и 6.

1) Реализовать классы для предварительной обработки (маскирования), вычисления точечных признаков (SIFT) 
   и разреженного оптического потока (calcOpticalFlowPyrLK()).
   В каждом классе должен быть метод инициализации (возможно, с передачей параметров алгоритма - winSize = (11, 11), 
   NFRAME = 10/120), метод обработки очередного изображения и метод сохранения отладочной информации.
   Отладочная информация - последовательность кадров, сохраняется классами, если у них включен 
   "отладочный режим" (включается отдельным методом, по умолчанию - выключен).

2) При практической реализации классы можно разрабатывать и использовать по очереди.

3) Чтобы замаскировать признаки, вычисляемые с помощью детектора SIFT, в классе предварительного маскирования
   предусмотреть методы маскирования полутонового изображения и массива точек.

4) В основной программе (из main()) нужно организовать цикл по выборке кадров из исходной видеопоследовательности,
   и передавать их по очереди в три объекта классов, которые реализуют алгоритмы обработки. У класса-оптического 
   потока включить отладочный режим.
   Основные этапы главного цикла:
     1) Получение очередного полутонового изображения img
     2) Маскирование img -> imgMasked
     3) Вызов класса оптического потока для изображения imgMasked
