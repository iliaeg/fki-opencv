Задание 6. 
Пример вычисления разреженного оптического потока методом Лукаса-Кэнада.
Визуальная оценка надежности прослеживания точечных признаков с помощью алгоритма разреженного оптического потока.

Пример вызова: https://docs.opencv.org/3.4/db/d7f/tutorial_js_lucas_kanade.html

1) Использовать задание 5) в качестве основы.
2) Сформировать массив координат точечных признаков SIFT.
3) Для этих точечных признаков вычислить разреженный оптический поток с заданной продолжительностью по числу кадров.
   Т.е., если NFRAMES = 5, то вычислить оптический поток для кадров (i, i+1, i+2, i+3, i+4).
   На i-м кадре в качестве точек prevPts взять массив координат SIFT-признаков.
   На кадрах (i+1, i+2, i+3, i+4) в качестве prevPts брать nextPts, полученные на предыдущем кадре.
   Использовать функцию calcOpticalFlowPyrLK(). Размер окрестности задать winSize = (11, 11).
   Для каждого точечного признака запомнить трек на изображении в виде последовательности NFRAMES точек.
4) На кадре (i+NFRAMES-1) построить запомненные треки - в виде набора отрезков, в конце последнего поставить маркер-окружность.
5) Каждый 10-й кадр с шага 4) записать в отдельный файл.
6) Обработку кадра (i+NFRAMES) начать с шага (2).
7) Повторить задание для NFRAMES = 10.
